# React 爬坑指南

作为一名主修后端的职业程序员，第一次接触 React 的感觉就像是：明明和人约好了要单挑，结果发现对方来了一大帮人，被华丽滴群殴了。

曾几何时，前端在我看来就是：html + css + js，不过如此嘛！

好吧，React 技术栈刷新了我对前端的认识。

水很深，坑很大，真不容易啊，故立此项目，记录爬坑点滴。
