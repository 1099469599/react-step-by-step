# react-step-by-step

## step06 antd

> **关键点**
>
> - `antd` - [官方教程](https://ant.design/docs/react/introduce-cn)

**运行方法**

- **运行开发环境**

```sh
$ npm install
$ NODE_ENV=dev # 如果是windows环境，应执行：set NODE_ENV=dev
$ npm start
```

- **运行生产环境**

```sh
$ npm install
$ NODE_ENV=prod # 如果是windows环境，应执行：set NODE_ENV=prod
$ npm start
```
