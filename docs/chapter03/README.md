# Chapter03 - Webpack

## :memo: 知识点

### Webpack

[Webpack](https://webpack.github.io/) 是一个模组打包工具（module bundler），它可以将许多模块封装成几个捆绑资源。

webpack 允许根据需要去加载应用程序的部件。使得 Javascript 应用可以高度复用。
