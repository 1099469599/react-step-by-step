# react-step-by-step

react-step-by-step 是一个从零开始学习 react 的教程。

## 说明

`docs` 目录遵循 [**GitBook**](https://github.com/GitbookIO/gitbook) 规范（Gitbook 配置选项在 book.json 中，可以按需修改），可以生成静态电子书网站，便于阅读。

### 运行条件

本地已安装 Node.js

### 运行方法

**安装gitbook-cli：**

```sh
$ npm install -g gitbook-cli
```

**生成静态资源文件：**

```sh
$ gitbook build
```

**启动电子书网站：**

```sh
$ gitbook serve
```

打开浏览器，访问 http://localhost:4000
